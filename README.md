# 云边端协同智能巡检与运维平台

## 项目简介

本项目面向智慧机房与实验室运维场景，构建一套云边端协同的智能巡检与运维平台。系统支持设备管理、数据采集、实时监控、异常告警、知识库问答、ROS2巡检机器人接入和AI视觉识别。
