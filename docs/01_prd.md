# 云边端协同智能巡检与运维平台 PRD 初稿

## 1. 产品定位

本产品面向智慧机房、实验室和小型数据中心运维场景，提供设备管理、数据监控、异常告警、智能问答、巡检任务和AI视觉识别等能力。

## 2. 用户角色

| 角色 | 说明 |
|---|---|
| 系统管理员 | 管理用户、角色、权限和系统配置 |
| 运维工程师 | 查看设备状态、处理告警、执行巡检任务 |
| 普通查看人员 | 查看设备状态和监控数据 |
| 设备维护人员 | 查看设备详情和维护记录 |

## 3. V1 阶段核心功能

| 模块 | 功能 |
|---|---|
| 用户与权限 | 登录、用户信息、角色权限 |
| 设备管理 | 设备新增、编辑、删除、查询 |
| 数据上报 | 模拟设备上报CPU、内存、磁盘、温湿度 |
| 监控大屏 | 展示设备统计和运行指标 |
| 告警管理 | 告警规则、告警记录、告警处理 |
| 部署 | Docker Compose 一键启动 |

## 4. V1.0.0 功能范围

V1.0.0 为项目初始化版本，不包含业务功能，只包含：

- 项目结构；
- 文档体系；
- Git 分支规范；
- README；
- CHANGELOG；
- 环境变量示例；
